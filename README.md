# Reclaim - AI Powered Lost and Found System

A full-stack MERN (MongoDB, Express, React, Node.js) application for managing lost and found items with AI-powered matching capabilities.

## 🚀 Project Structure

This project follows a monorepo structure with separate frontend and backend directories:

```
Reclaim/
├── backend/          # Node.js/Express API server
├── frontend/         # React application
├── PROJECT_STRUCTURE.md  # Detailed folder structure documentation
└── SETUP_GUIDE.md       # Setup and installation guide
```

## 📋 Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- MongoDB (local installation or MongoDB Atlas account)

## 🛠️ Quick Start

### Backend Setup
```bash
cd backend
npm install
# Copy env.example.txt to .env and configure
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
# Copy env.example.txt to .env and configure
npm run dev
```

## 📚 Documentation

## 🏗️ Tech Stack

- **Frontend**: React 18, Vite, React Router
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT (JSON Web Tokens)

## 📝 Development

- Backend runs on: `http://localhost:5000`
- Frontend runs on: `http://localhost:3000`
