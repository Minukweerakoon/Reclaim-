# Lost & Found Smart Assistant - User Guide

## 🎯 Quick Start

### Reporting a Lost Item

1. **Start the conversation**
   - Type: "I lost my [item]"
   - Example: "I lost my black iPhone 15"

2. **Provide details**
   - The AI will ask follow-up questions
   - Mention: color, brand, location, time
   - Be specific for better matching

3. **Add a photo** (optional but recommended)
   - Click "📷 Take Photo" to use your camera
   - Or click "📁 Upload File" to select from device
   - Ensure good lighting and clear focus

4. **Add voice description** (optional)
   - Click the microphone button
   - Speak clearly about unique features
   - Max 120 seconds

5. **Submit**
   - Review the validation summary
   - System checks: text quality, image detection, voice consistency
   - Overall confidence score displayed

---

## 📱 Using the Camera

### Taking a Good Photo

**Do:**
- ✅ Use good lighting (natural light is best)
- ✅ Get close to the item
- ✅ Keep the item in focus
- ✅ Show unique features (scratches, stickers, etc.)
- ✅ Use a plain background

**Don't:**
- ❌ Take blurry photos
- ❌ Use poor lighting
- ❌ Include personal information (IDs, addresses)
- ❌ Take photos from too far away

### Camera Controls

- **Switch Camera**: Tap the camera switch icon (mobile only)
- **Retake**: Click "Retake Photo" if not satisfied
- **Confirm**: Click "Use This Photo" when ready

---

## 🎤 Voice Recording Tips

1. **Speak clearly** - Enunciate your words
2. **Mention details** - Describe unique features
3. **Be concise** - Max 120 seconds
4. **Quiet environment** - Reduce background noise

**Example:**
> "It's a brown leather wallet with a scratch on the corner. It has my university ID card inside. I think I left it in the library around 2pm today."

## ✅ Understanding Validation Results

### Confidence Scores

- **90-100%**: Excellent - All checks passed
- **70-89%**: Good - Minor inconsistencies
- **50-69%**: Fair - Some concerns
- **Below 50%**: Poor - Needs improvement

### What Gets Checked

1. **Text Quality** (completeness, coherence)
2. **Image Detection** (object found, confidence)
3. **Voice Consistency** (matches text/image)
4. **Cross-Modal Alignment** (all inputs agree)

## 🔧 Troubleshooting

### "No objects detected"
- **Solution**: Retake photo with better lighting
- Ensure item is clearly visible
- Try different angle

### "Image too blurry"
- **Solution**: Hold camera steady
- Tap screen to focus (mobile)
- Use better lighting

### "Voice transcription failed"
- **Solution**: Speak more clearly
- Reduce background noise
- Check microphone permissions

### "File too large"
- **Solution**: Maximum file size is 10MB
- Compress image before uploading
- Use camera feature instead

### Camera not working
- **Solution**: Check browser permissions
- Use HTTPS (required for camera access)
- Try different browser (Chrome/Safari recommended)

## 💡 Best Practices

### For Lost Items

1. Report as soon as possible
2. Include photo if you have one from before
3. Mention last known location and time
4. Describe unique identifying features
5. Check back regularly for matches

### For Found Items

1. Take clear photos immediately
2. Note exact location and time found
3. Describe the item's condition
4. Mention any identifying marks
5. Keep item safe until claimed

## 🔒 Privacy & Security

- **Photos**: Processed locally, not permanently stored
- **Voice**: Transcribed and deleted after processing
- **Personal Info**: Avoid including sensitive data in photos
- **Data**: Validation results stored for matching only
